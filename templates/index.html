Flask API application is used to test (predict, recognize) a trained neural network model - CNN - 6-class image classifier.
The machine learning project concerns classification - images divided into 6 classes-categories (buildings, forest, glacier, mountain, sea, street). The neural network architecture used was built in a sequential convention. The CNN-based model (architecture and weights) was built during training and saved to a file. Based on Flask technology, two Jinja2 templates were rendered, two endpoints were created ("/", /predict).
